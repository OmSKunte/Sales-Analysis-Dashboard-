# 📊 Sales Analysis Dashboard

An interactive Power BI dashboard built to analyze sales performance of Micro Form Tools.  
It provides clear insights into revenue, product performance, and trends using visual analytics.

---

## 🚀 Features

- 📈 Year-To-Date (YTD) Sales Analysis
- 📊 Quarter-To-Date (QTD) Sales Tracking
- 📅 Monthly & Weekly Sales Trends
- 🏆 Top 5 Products by Sales & Reviews
- 🧩 Product Category Breakdown
- 🎯 Dynamic Filters for better analysis
